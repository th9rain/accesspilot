—— 严重缺乏安全防火墙

Elva 老白的核心法则

守护银发数字晚年，安全与便利缺一不可

CORE COMMANDMENT

原始屏幕留在端侧，
声导规划飞上云端。

手机截图、完整 OCR、身份证号、手机号、验证码、付款信息等极其敏感数据默认绝对不上传云端。端侧先完成页面观察、隐私去敏感化、风险初判；仅在任务逻辑复杂、需要多步导航规划时，才把脱敏后的结构化页面描述送给大模型，并由本地翻译为语音“声导”输出。

2. 隐私防火墙

Screen Observer 提取UI元素；Privacy Firewall 对PII个人敏感数据无感脱敏，计算 `cloud_safe` 状态。

阻断未脱敏数据上云
Layer 03

3. 本地路由决策

Local Router 判断风险及复杂度。低风险如调大字号本地解决；高风险如扣款直接拦截；复杂业务才升级上云。

Gemma 端侧分流
Layer 04

4. 云端规划思考

Cloud Planner (Gemma-Cloud) 基于结构化脱敏数据进行多步规划逻辑推理，输出可操作节点建议。

RAG知识增强 Rerank
Layer 05

5. 安全执行与声导

Safety Guard 最终过滤。OpenClaw 仅在白名单下工作。TTS 转换为温馨的声音实时对老人声导指引。

代码级数据实体：Elva 老白五大核心模型

强类型契约设计，保证 Agent 链路各环节的输入输出有迹可循、坚不可摧

01

EdgeEvent

端侧触发事件。描述系统捕获事件类别、触发置信度、来源组件以及数据是否受本地约束锁定。

Event Collector 发出
02

ScreenObservation

屏幕观察脱敏结果。包含设备布局结构、脱敏字段字典、安全因子判定位及关乎上云的 `cloud_safe` 开关。

最关键的隐私边界线
03

RoutingDecision

本地路由决策。判定本地响应、拒绝上云还是启动云端大模型协作，将不需要云端理解的任务分流在本地。

Local Router 控制层
04

NextAction

下一步动作规范。云端基于脱敏上下文下发的语义级任务操作，包括动作定位、参数要求、解释以及风险系数评估。

Cloud Planner 发出
05

GuardDecision

安全过滤器决策。针对高风险交易或敏感权限做终极校验，直接执行 Allow、Require_Confirmation 或 Deny 并降级播报。

执行组件的终极门锁

端云协同分工：Gemma 双模型合理协同机制

最大程度挖掘小模型端侧独立运行优势与云端模型深度推理潜能

Gemma-2B 端侧极低延迟引擎

本地运行 · 绝无网路损耗 · 隐私零风险

负责执行本地无延迟的极速响应任务，以及核心安全敏感的动作。

  • 敏感实体剔除： 本地发现并隐藏身份证号码、真实手机、动态OTP。
  • 极低耗电触发判定： 判断是否需要引导用户或弹出老白气泡。
  • 低风险设置跳转： 独立指挥 Accessibility 跳转并标红调大字号项。

Gemma-Cloud 复杂逻辑规划大脑

云端 RAG 推理 · 复杂流程分析 · 柔性规划

仅在端侧 `cloud_safe=true` 的脱敏结构数据到达后，进行跨业务、高信息密度的多步规划思考。

  • 复杂政务逻辑理解： 跨多张页面，结合 RAG 提示老人当前流程在办什么。
  • 暖心协助简报生成： 理解复杂的页面报错并提炼为脱敏温馨的家人协助摘要。
  • 受控动作流程推荐： 基于行为和全局场景，规划下一步最佳推荐点击按钮。

Elva 老白：清晰的阶梯式工程集成路线图

从高度确定性 V1 原型，到结合 OpenClaw 的真实系统集成

PHASE 01 (DONE)

V1: 确定性沙盒原型

跑通前端演示程序。定型 5 大类核心数据对象结构及 4 个高保真 Demo Scenarios 逻辑闭环。

PHASE 02

V2: 云端 Planner 替换

接通云侧 Gemma-Cloud。通过 Schema 校验与 RAG 外部合规政策数据库连通，生成智能规划。

PHASE 03

V3: Android 无障碍框架

在 Android 侧集成 AccessibilityService 与通知拦截层。实现真实提取屏幕物理元素及防脱敏泄漏。

PHASE 04

V4: 端侧 Gemma 本地化

通过本地硬件加速在端侧运行轻量级小模型。实现独立脱敏拦截，网络离线依然安全无阻。

PHASE 05

V5: OpenClaw 受控闭环

对接 OpenClaw，允许在安全白名单内对设备执行 Click、Type、Scroll、Summarize 动作。

真实开发路径： 1. 模拟 Planner → 2. Gemma 云端规划大模型 → 3. Android 无障碍隐私网关 → 4. 端侧 Gemma 本地化 → 5. OpenClaw 白名单执行 步步为营，安全可靠
